# OpenClaw 中文流量站（Next.js）项目交接文档

## 1. 项目要做什么
这个项目的目标是做一个 **OpenClaw 中文 SEO 获客站**，通过“教程内容 -> 微信咨询 -> 付费服务”跑通最小商业闭环。

核心闭环：
1. 用户在搜索引擎搜 `openclaw 安装/部署/报错` 等长尾词。  
2. 进入站内教程页，解决实际问题并建立信任。  
3. 通过唯一 CTA（加微信）进入私域咨询。  
4. 转化为远程安装、排错、部署交付等服务。

## 2. 当前已完成（已可运行）
已在 `openclaw-site` 完成 Next.js MVP：
1. 首页：定位、痛点、内容结构、CTA。  
2. 教程列表展示。  
3. 动态详情页路由：`/guides/[slug]`。  
4. 文章数据源：`src/data/posts.ts`（当前4篇示例）。  
5. 基础 SEO metadata（站点级 + 文章级）。  
6. 样式与移动端基础适配。  
7. 通过验证：`npm run lint`、`npm run build`。

## 3. 目录与关键文件
1. `src/app/page.tsx`：首页。  
2. `src/app/guides/[slug]/page.tsx`：文章详情页（SSG）。  
3. `src/data/posts.ts`：文章内容数据。  
4. `src/app/layout.tsx`：全局 metadata。  
5. `src/app/globals.css`：全局样式。

## 5. 内容生产策略（30天）
1. 第1周：基础教程 5 篇（安装/部署）。  
2. 第2周：报错排查 7 篇（按日志关键词命名）。  
3. 第3周：实战案例 5 篇。  
4. 第4周：进阶优化 + FAQ 5 篇。  

每篇文章固定结构：
1. 问题场景（用户搜索原句）。  
2. 原因分析。  
3. 可复制步骤。  
4. 常见坑。  
5. 验证结果。  
6. CTA（加微信/提交表单）。

## 6. KPI（用于判断是否继续放大）
30天建议目标：
1. 收录页数 >= 15。  
2. 自然搜索点击 >= 300。  
3. 微信/表单咨询 >= 20。  
4. 有效线索率 >= 30%。  
5. 首单 >= 2。
